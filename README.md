## Deployment

This landing page is deployed on Vercel.

### Deploy to Vercel

1. Install Vercel CLI:
```bash
npm i -g vercel
```

2. Deploy:
```bash
vercel
```

3. For production:
```bash
vercel --prod
```

## Local Development

Simply open `index.html` in your browser or use a local server:

```bash
npx serve
```

## Update APK Link

To update the download link, edit the `href` attribute in `index.html`:

```html
<a href="YOUR_NEW_APK_LINK" class="download-btn" download>
```
